Practical regime

                

                  Quant desk view on live MT5 GET /api/snapshot: a trader cockpit (live gauges + regime vs
                  strategy vs action), optional strategy_key so the API returns a practical_focus payload (one
                  scorecard row + matching selector scores + regime count for that id only), gate checklist, history charts, deltas vs
                  last response in this tab, and honest limits where the wire has no transition matrix.
                

              


              

                

                  

                    Method
                    GET
                  

                  

                    Path
                    /api/snapshot
                  

                  

                    Full URL
                    
                  

                  

                    API base
                    
                  

                

              


              

                Request — same layout idea as Postman (params + raw body)
                

                  This is a GET: there is no POST body. The params grid and raw JSON below are two
                  views of the same query object (edit either). Send calls GET /api/snapshot.
                  Include regime_id plus strategy_key (e.g. R07-S01) so the server returns a
                  practical_focus payload: one scorecard row, aligned selector scores, and regime bar count for that id only.
                

                

                  
                  
                  
                

                

                  

                    
Params

                    
Query string fields (editable).

                    

                  

                  

                    
Raw (JSON)

                    
Mirrors params; sent as query string, not as a request body.

                    
                  

                

                
Request preview

                

                

                  
                  
                  
                  
                

              


              

                
Response

                


                

                  

                    When strategy_key matches a row under regime_detail.strategies, the server returns
                    snapshot_scope: practical_focus (narrower JSON). Everything here is still that single live MT5 pull —
                    no fabricated fills. Money metrics that need a ledger are called out in panel 5b. Deltas use
                    sessionStorage from the prior successful send in this tab. With Live refresh enabled in the
                    request toolbar, the same query runs on an interval while this page section is open.
